# NGSidekick

Tools for neuroglancer scenes.


## Installation

- Using uv:

```bash
uv add ngsidekick
# or in an existing environment
uv pip install ngsidekick
```

## Development

Create an environment and run tests:

```bash
uv venv
uv pip install -e .[test]
pytest
```
